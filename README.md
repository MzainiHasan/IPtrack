> Tool sederhana untuk melacak informasi lokasi IP address menggunakan API dari [ipwhois.io](https://ipwhois.io)

## 🔧 Cara Penggunaan

```bash
python iptracker.py 8.8.8.8
```

## 🎯 Output

- Kota, Negara, Ibukota
- Benua, Tipe IP, Koordinat
- Kode negara dan telepon
